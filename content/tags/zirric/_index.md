Early development, experimental programming language with an implicit but strong and dynamic type system.

Development is in an on-off phase. Docs can be found at [zirric.knabel.dev](https://zirric.knabel.dev), the source code is available at [code.knabel.dev/zirric-lang/zirric](https://code.knabel.dev/zirric-lang/zirric).
A development name for Zirric was Blush, which might still be used in some places.
